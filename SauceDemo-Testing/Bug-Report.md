# Bug Report

## Bug Title
Cart badge does not update after removing a product

## Environment
- Browser: Chrome
- Version: Latest
- OS: Windows 11

## Steps to Reproduce
1. Login with valid credentials
2. Add a product to the cart
3. Open the cart
4. Remove the product

## Expected Result
The cart badge should display 0 items.

## Actual Result
The cart badge still displays 1 item.

## Severity
Medium

## Priority
Medium
