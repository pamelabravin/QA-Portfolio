# Test Plan - Amazon Login Testing

## Objective

Validade the functionality of the Amazon login feature and ensure users can authenticate successfully.

## Scope

The following features will be tested:

- User Login
- Email Validation
- Password Validation
- Error Messages
- Password Visibility Toggle

## Out of Scope

- User Registration
- Payment Processing
- Order Management

## Test Types

  - Functional Testing
  - Exploratory Testing
  - Negative Testing
 
## Test Environment

  - Browser: Google Chrome
  - Device: Desktop
  - Operating System: Windows 11
 
## Entry Criteria

  - Login page is available
  - Test accounts are created

## Exit Criteria

  - All planned test cases executed
  - No critical defects remain open

# Risks

  - Unavailable test environment
  - Invalid test data

# Deliverables

  - Test Cases
  - Bug Reports
  - Exploratory Testing Report
  - Test Summary Report
