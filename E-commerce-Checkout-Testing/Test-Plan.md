# E-commerce Checkout Test Plan

## Objective

Validate the checkout process of an e-commerce website.

## Scope

- Product selection
- Shopping cart
- Address validation
- Payment processing
- Order confirmation

## Out of Scope

- Inventory management
- Admin panel

## Test Types

- Functional Testing
- Exploratory Testing
- Negative Testing

## Risks

- Payment gateway failures
- Incorrect order totals
